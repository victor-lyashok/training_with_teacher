Учебный проект, посвященный применению простейших методов обучения с учителем.

Цель проекта: выбор лучшей модели, прогнозирующей вероятность ухода клиентов банка «Бета-Банк».

Данные: исторические данные о клиентах банка «Бета-Банк» со следующими признаками:
* 	RowNumber — индекс строки в данных
* 	CustomerId — уникальный идентификатор клиента
*	Surname — фамилия
* 	CreditScore — кредитный рейтинг
* 	Geography — страна проживания
* 	Gender — пол
* 	Age — возраст
* 	Tenure — сколько лет человек является клиентом банка
* 	Balance — баланс на счёте
* 	NumOfProducts — количество продуктов банка, используемых клиентом
* 	HasCrCard — наличие кредитной карты
* 	IsActiveMember — активность клиента
* 	EstimatedSalary — предполагаемая зарплата
*	Exited — факт ухода клиента

Результаты проекта: была найдена лучшая модель (случайный лес с увеличением выборки, максимальной глубиной 25 и количеством деревьев 80), которая дает значение f1 равное 0,5913. Показано, что наиболее важными признаками при прогнозировании оказываются (в порядке убывания) возраст, предполагаемая зарплата, кредитный рейтинг, количество продуктов банка, используемых клиентом.
